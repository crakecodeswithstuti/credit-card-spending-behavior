Abstract

This project focuses on analyzing credit card transaction data in India to understand customer spending behavior. With the increasing use of digital payments, analyzing transaction data helps identify trends, patterns, and user preferences.

The dataset contains details such as transaction amount, category of spending, location, and transaction type. Tableau was used for data visualization and analysis.

Exploratory Data Analysis (EDA) techniques were applied to identify spending patterns across different categories, cities, and time periods. The analysis highlights major spending sectors, peak transaction periods, and customer behavior trends.

The results help businesses, banks, and financial institutions make better decisions related to marketing strategies, fraud detection, and customer targeting.[file:1][file:2]

Introduction

With the rapid growth of digital payments, credit cards have become a major mode of financial transactions. Every transaction generates valuable data that can be analyzed to understand consumer behavior.

Analyzing credit card data helps in identifying spending patterns, popular categories, and regional trends. This information is useful for banks, businesses, and policymakers.

This project uses Tableau to analyze credit card transaction data in India and extract meaningful insights using data visualization techniques.[file:1]
